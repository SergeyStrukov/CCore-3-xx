This part provides two Class-functions for the range scanning and the string scanning. 


/* struct ScanRange<R> */

template <class R>
struct ScanRange
 {
  R before;
  R next; // starts at element satisfies pred

  template <class Pred>
  ScanRange(R r,Pred pred);
 };

/* struct ScanStr */

struct ScanStr : ScanRange<StrLen>
 {
  using ScanRange<StrLen>::ScanRange;

  ScanStr(StrLen str,char ch);
 };

ScanRange scans the given range forward in the search of elements, satisfying the given predicate. Once such element is found, it splits the range on two parts: before and next. next starts at the found element. If no required elements has been found, then before sets to the full range and next to the empty.

ScanStr is the ScanRange for the StrLen type with one additional constructor: it takes the character to search it in the string.
