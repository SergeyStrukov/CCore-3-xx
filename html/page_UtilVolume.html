Volume

Volume is a simple utility to work with volumes. 


CCore-Volume.exe -a <src-dir> [<vol-file>]

CCore-Volume.exe -l <vol-file>

CCore-Volume.exe -x <vol-file> [<dst-dir>]

First, you can create a volume. You must provide a directory with files. These files will be placed in a volume file. The name of each file is "/file" or "/dir/file". If the volume file name is not given, the name of the directory with the appended extension ".vol" is used.

You can list volume files.

Finally, you can extract files from a volume into some directory. If the directory name is not given, the name of the volume file is used with skipped extension.
