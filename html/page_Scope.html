Scope

Files CCore/inc/Scope.h CCore/src/Scope.cpp

Scope is a scope-marker. A local variable of this type can be used to assign a name to the current scope. 


class Scope : NoCopy
 {
   ....

  public:

   explicit Scope(StrLen name); // persistent

   ~Scope();

   template <class P>
   static bool Print(P &&out);
 };

Print() prints scope names from the most recent, separated by the space. The return value is true, iff at least one scope name is defined.
