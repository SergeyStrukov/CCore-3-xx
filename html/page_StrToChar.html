StrToChar

Files CCore/inc/StrToChar.h CCore/src/StrToChar.cpp

This file contains two classes to convert Char text into char text and vise versa. 


/* class StrToChar */

class StrToChar : NoCopy
 {
   ....

  public:

   explicit StrToChar(StrLen text);

   ~StrToChar();

   PtrLen<const Char> operator + () const;

   operator PtrLen<const Char>() const;
 };

/* class CharToStr */

class CharToStr : NoCopy
 {
   ....

  public:

   explicit CharToStr(PtrLen<const Char> text);

   ~CharToStr();

   StrLen operator + () const;

   operator StrLen() const;
 };

In UTF8-compatible mode these classes allocates a memory and performes the required conversion between Unicode and UTF8 representations. An exception is throwns in case of broken UTF8 sequence.

In ASCII mode these classes are trivial.
