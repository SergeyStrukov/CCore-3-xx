(XCore) BinaryFile

Files CCore/inc/BinaryFile.h CCore/src/BinaryFile.cpp

XCore BinaryFile is similar to HCore one, but built on the AsyncBinaryFile class. 


class BinaryFile : public AsyncBinaryFile
 {
  public:

   // constructors

   BinaryFile() noexcept;

   explicit BinaryFile(StrLen file_name,FileOpenFlags oflags=Open_ToWrite);

   ~BinaryFile();

   // methods

   void open(StrLen file_name,FileOpenFlags oflags=Open_ToWrite);
 };

To open a file the DefaultFileDevice is used.
