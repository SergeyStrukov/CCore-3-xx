SymCount

This file provides simple tools to count symbols in a text.

SymCount()

SymCount() counts the number of symbols in a text string. In UTF8-compatible mode it differs than the number of characters. 


ulen SymLen(StrLen text);

SymDetector

SymDetector detects symbols in a character stream. 


class SymDetector : NoCopy
 {

  public:

   SymDetector() { reset(); }

   void reset();

   bool put(char ch);
 };

reset() resets the detector to the initial state.

put() puts the character into the detector, the return value is true iff the character completes the symbol.

SymCounter

SymCounter counts the number of symbols over the character stream. 


class SymCounter : NoCopy
 {
   ulen count;
   SymDetector det;

  public:

   SymCounter() : count(0) {}

   operator ulen() const { return count; }

   void put(char ch);

   void put(char ch,ulen len);

   void put(const char *str,ulen len);
 };

put() methods feed the counter by characters.

To get the current number of complete symbols of the stream cast to the type ulen.
