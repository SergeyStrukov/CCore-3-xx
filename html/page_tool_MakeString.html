MakeString

MakeString is a simple utility, it converts a text file into a file with C-string, which match the content of the original file. 


CCore-MakeString <input-file> <output-file>

CCore-MakeString -s <input-file> <output-file>

If you specify the option -s, C-style comments will be stripped from the result.

The output looks like 

"\n"
"int Version = 1 ;\n"
"\n"
" \n"
"\n"
"type Bool = uint8 ;\n"
"\n"
"Bool True = 1 ;\n"
"\n"
"Bool False = 0 ;\n"
"\n"
"type VColor = uint32 ;\n"
"\n"
"VColor NoColor = 0FFFFFFFFh ;\n"
"\n"
"type Coord = sint32 ;\n"
"\n"
"struct Point\n"
" {\n"
"  Coord x;\n"
"  Coord y;\n"
" };\n"
" \n"
"struct Ratio\n"
" {\n"
"  Coord a;\n"
"  Coord b;\n"
" }; \n"
"\n"
" \n"
