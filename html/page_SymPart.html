SymPart

Files CCore/inc/SymPart.h CCore/src/SymPart.cpp

This part contains functions to cut string parts, based on symbol (not character!) counts. All these functions are safe, i.e. return the maximum possible result. 


StrLen SymPrefix(StrLen str,ulen len);

StrLen SymPart(StrLen str,ulen off);

StrLen SymPart(StrLen str,ulen off,ulen len);

SymPrefix() cuts the string prefix with the given symbol length.

SymPart(ulen) cuts the string part, starting from the given symbol.

SymPart(ulen,ulen) cuts the string part, starting from the given symbol and with the given symbol length.
