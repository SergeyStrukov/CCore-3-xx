(XCore) DefaultFileDevice

Files CCore/inc/DefaultFileDevice.h CCore/src/DefaultFileDevice.cpp

XCore has a global default file device name. By default, it is "host". You can get or set this name by the following functions: 


void SetDefaultFileDevice(StrLen dev);

ulen CopyDefaultFileDevice(char buf[MaxPathLen]);

The first function sets the name. The name length must not exceed MaxPathLen, otherwise an exception is thrown.

The second function makes a copy of the name in the provided buffer. The buffer length must be MaxPathLen. The name length is returned.

Both functions are thread-safe.

The following helper can be used to simplify the name extraction. 


struct GetDefaultFileDevice : NoCopy
 {
  char buf[MaxPathLen];
  ulen len;

  GetDefaultFileDevice() { len=CopyDefaultFileDevice(buf); }

  operator StrLen() const { return Range(buf,len); }
 };
