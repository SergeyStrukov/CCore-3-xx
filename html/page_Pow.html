These two functions calculates the power of the first arguments. The second arguments is the power exponent. It must be an unsigned integral value. The function Pow_pos() assumes the exponent is positive. Otherwise the first argument is returned. The function Pow() assumes that T(1) is the one value for this type. 


/* Pow_pos() */

template <UIntType UInt,class T>
T Pow_pos(T a,UInt deg); // deg>0

/* Pow() */

template <UIntType UInt,class T>
T Pow(T a,UInt deg);
