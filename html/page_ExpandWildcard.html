template <FuncInitArgType<StrLen> FuncInit>
auto ExpandWildcard(FileSystem &fs,StrLen path,FuncInit func_init);

ExpandWildcard() calls the given functor on a file set, defined by file name with wildcards. Only wildcards in the file name are expanded. Functor is applied on each existing file, matching the given wildcard name.
