FeedBuf is a Class-function. 


template <class T>
struct FeedBuf
 {
  ulen delta;
  CmpResult cmp;

  bool overflow() const { return cmp<0; } // src has extra data

  bool underflow() const { return cmp>0; } // dst has extra space

  FeedBuf(PtrLen<T> dst,PtrLen<const T> src);
 };

It copies data from one range to another. The result value has two fields: delta and cmp.

delta is a number of object copied, it equlas the minimum of lengthes of source and destination ranges.

cmp if the result of comparison of the destination range length and the source ones. 


template <class T>
bool /* overflow */ Feedup(PtrLen<T> &dst,PtrLen<const T> src);

This function feeds the destination range from the source and updates dst to the remaining range. The overflow flag is returned. 


template <class T>
bool /* underflow */ Pumpup(PtrLen<T> &dst,PtrLen<const T> &src);

This function pumps the destination range from the source and updates both dst and src. The underflow flag is returned.
