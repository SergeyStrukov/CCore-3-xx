LaunchPath

Files CCore/inc/LaunchPath.h CCore/src/LaunchPath.cpp

This class obtains the full path of the running executable file. 


class LaunchPath : NoCopy
 {
   char buf[MaxPathLen+1];
   StrLen path;

  public:

   LaunchPath();

   StrLen getPath() const { return path; }

   StrLen getDir() const { return PrefixPath(path); }
 };

Constructor obtains the path. An exception is thrown in case of error.

getPath() returns the full path to the executable file.

getDir() returns the full path to the executable directory.

This class is built upon the target class Sys::LaunchPath, declared in the header sys/SysLaunchPath.h. 


struct LaunchPath
 {
  StrLen path;
  ErrorType error;

  explicit LaunchPath(char buf[MaxPathLen+1]);
 };
