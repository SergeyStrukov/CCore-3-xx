OptMember

Files CCore/inc/OptMember.h CCore/src/OptMember.cpp

OptMember is a container class for a single object of the given type. The memory for the object is reserved inside. 


template <class T>
class OptMember : NoCopy
 {
   ....

  public:

   OptMember();

   ~OptMember();

   // object ptr

   T * operator + () const;

   bool operator ! () const;

   T * getPtr_unsafe() const;

   T * getPtr() const;

   T & operator * () const;

   T * operator -> () const;

   // create/destroy

   template <class ... SS>
   void create(SS && ... ss);

   void destroy();
 };

The default constructor creates an empty object.

create() creates the object inside, using the provided arguments for the constructor. If the object already exists, this method has no effect.

destroy() destroys the object inside. If the object does not exist, this method has no effect.

To access the object the Object Pointer Interface is implemented.

getPtr_unsafe() must be used only if the container is full. Other access methods throw an exception if the container is empty.
