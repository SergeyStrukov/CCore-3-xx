CharUtils

Files CCore/inc/CharUtils.h CCore/src/CharUtils.cpp

This file contains simple tools to work with text, represented by a Char sequence. In UTF8-compatible mode conversion from char text into Char text is performed.

struct FillCharBuf
 {
  ulen len;

 };

This Class-function fills a Char buffer with a text. The result fields is the output length and the overflow flag.
